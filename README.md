# Dark Truth

An action adventure game built with Unity, playable directly in your browser.

**[Play it on itch.io](https://abdullahhamdan.itch.io/dark-place)**


## About
A game where you need to investigate and find a way to clear the level, each level has different purpose!

 
---
 
## Built With
 
- [Unity](https://unity.com/) — game engine
- C# — scripting language
- Using brackeyes art assets
